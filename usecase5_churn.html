Churn Risk Scoring Agent
Agent Actions:
• Scores churn risk using ML model across all customers
• Sizes gap by segment and value tier (High/Med/Low)
• Identifies drivers: declining engagement, longer recency, price sensitivity
Output:
Risk-scored customer list with churn drivers and value tiers
Intervention Design Agent
Agent Actions:
• Recommends save-offers tiered by customer value
• Designs win-back journey sequences
• Proposes service outreach for highest-value at-risk
Output:
Save playbook with offer tiers, journeys, and outreach list for approval
Save Campaign Agent
Agent Actions:
• Launches approved interventions via CRM platform
• Tracks save rate and offer redemption in real-time
• Escalates non-responders for human follow-up
Output:
Daily save metrics, non-responder escalation list, optimization suggestions
Churn Measurement Agent
Agent Actions:
• Calculates churn reduction vs target (20%)
• Measures ROI of save offers by tier
• Identifies patterns in saved vs churned customers
Output:
Churn reduction report, ROI analysis, playbook recommendations
